Heatmaps of Parking Violations and Predicted Services Needed

These heatmaps display parking violations alongside predicted service requests.

The prediction model, built with TensorFlow, uses key features such as violation_count, land_av, total_av, and ACRES to forecast request_count.

The model architecture includes a Sequential model with 64 neurons in the first layer and is trained over 10 epochs using an 80/20 training split.

These visualizations provide insight into predicted request patterns, enabling city officials to anticipate service demands and allocate resources efficiently.
